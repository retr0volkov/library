РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
"Библиотечный каталог технической литературы"

1. Введение
Приложение предназначено для просмотра каталога технической литературы по курсу "Программирование сетевых приложений".

2. Основные функции
- Просмотр списка книг
- Сортировка по названию, автору, году издания
- Поиск по каталогу
- Открытие ссылок на полные тексты книг

3. Как пользоваться
Для загрузки каталога выберите в меню "Файл" -> "Загрузить каталог".
Для открытия руководства выберите "Справка" -> "Открыть руководство (PDF)".
